# WAI — Worked All Ireland

## Kaj je WAI?

Worked All Ireland (WAI) je radioamaterski program, ki pokriva celoten irski otok — tako Republiko Irsko (predpona EI) kot Severno Irsko (predpona GI). Program temelji na irskem koordinatnem sistemu (Irish Grid Reference System) in razdeli otok na kvadrate, ki sluzijo kot reference za radijske aktivacije.

S priblizno 400 referencami WAI ponuja pregleden, a privlacen nabor aktivirajocih obmocij. Program je posebej omembe vreden, ker premoscuje politicne meje in celoten otok obravnava kot eno radijsko-geografsko enoto. Radioamaterji iz obeh delov drzave skupaj delajo na tem programu, kar poudarja povezovalno naravo radioamaterstva.

WAI je namenjen tako irskim radioamaterjem, ki zelijo sistematicno aktivirati svojo domovino, kot tudi mednarodnim postajam, ki zelijo zbirati irske reference. Podezelska struktura velikih delov Irske naredi prenosne aktivacije posebej privlacne, saj so mnogi kvadrati dosegljivi le z namenskimi ekspedicijami.

## Pravila

- **Najmanj QSO:** Ni fiksnega minimuma za veljavno aktivacijo
- **Aktivacijska cona:** Znotraj ustreznega irskega kvadrata
- **Oprema:** Brez omejitev, dovoljene vse stopnje moci
- **Nacini delovanja:** Vsi obicajni nacini (SSB, CW, FM, digitalni nacini)
- **Posebne zahteve:** WAI-referencna stevilka mora biti pravilno navedena; QSL-kartice ali elektronske potrditve kot dokaz

## Tocke & Nagrade

| WAI Silver | 100 razlicnih WAI-kvadratov delanih |
| WAI Gold | 200 razlicnih WAI-kvadratov delanih |
| WAI All Ireland | Vse razpolozljive WAI-reference delane |
| WAI Activator Award | Doloceno stevilo WAI-kvadratov samih aktiviranih |

## Kako zaceti

1. Pridobi informacije na uradni strani WAI in prostudiri seznam referenc
2. Doloci svoj irski kvadrat ali izberi ciljno obmocje za aktivacijo
3. Opravi zveze QSO z navedbo WAI-reference
4. Oddaj dnevnike in dokumentiraj delane kvadrate
5. Sistematicno zbiraj nadaljnje kvadrate — posebej redke podezelske reference so iskane
