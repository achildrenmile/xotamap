# WAI — Worked All Ireland

## What is WAI?

Worked All Ireland (WAI) is an Irish amateur radio programme based on Irish Grid reference squares. The goal is to work or activate as many different Irish grid references as possible. Approximately 400 references.

## Rules

- **Minimum QSOs:** 1 contact
- **Activation zone:** Within the grid square
- **Equipment:** Any setup
- **Region:** Ireland
